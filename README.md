Zamonaviy, tezkor va ko'p funksiyali rang palitrasi generatori. Dasturchilar va dizaynerlar uchun mukammal loyihalashtirilgan yechim.

## ✨ Xususiyatlar (Features)

- **Har xil generatsiya rejimlari**: Tasodifiy (Random), Monochromatic, Analogous, Triadic, Tetradic.
- **Tezkor klavishlar**: `Space` tugmasi orqali yangi palitra yaratish.
- **Rang qulflash**: Yoqtirgan ranglaringizni qulflang va saqlab qoling (`Lock/Unlock`).
- **Nusxa olish**: Birgina chertish orqali ranglarni HEX, RGB hamda HSL ko'rinishida nusxalash.
- **Xotira va Tarix (Undo & Save)**: Oldingi natijalarni qaytarish va yoqtirgan palitralarni saqlash funksiyasi.
- **Eksport (Export Code)**: Tayyor ranglarni CSS Variables, SCSS yoki Tailwind Config qilib eksport qilish.
- **Zamonaviy UI**: Motion animatsiyalar, toza dizayn va mukammal kontrast (Dark/Light text adjust).

## 🛠 Texnologiyalar

- [React 19](https://react.dev/) - UI kutubxonasi
- [Vite](https://vitejs.dev/) - Tezkor bould tizimi
- [Tailwind CSS 4](https://tailwindcss.com/) - Stilizatsiya
- [Chroma.js](https://gka.github.io/chroma.js/) - Ranglar bilan ishlash
- [Motion](https://motion.dev/) - Silliq animatsiyalar
- [Lucide React](https://lucide.dev/) - Zamonaviy ikonalar

## 🚀 O'rnatish

Loyihani o'z kompyuteringizda ishga tushirish uchun quyidagi bosqichlarni bajaring:

1. Repozitoriyni yuklab oling yoki klon qiling:
```bash
git clone https://github.com/sanjarbek404/Palitra-Pro.git
```

2. Loyiha papkasiga o'ting:
```bash
cd rang-palitrasi
```

3. Bog'liqliklarni o'rnating:
```bash
npm install
```

4. Dasturni ishga tushiring:
```bash
npm run dev
```

Loyiha mahalliy serverda (masalan, `http://localhost:3000`) ishga tushadi.
